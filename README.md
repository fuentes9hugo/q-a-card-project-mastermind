# q-a-card-project-mastermind
## Descripción

Os presentamos un nuevo reto práctico para seguir afianzando vuestros conocimientos en HTML y CSS.

La idea es transformar el diseño que tenéis en la plataforma en un proyecto 100% funcional. Vais a construir una tarjeta interactiva de preguntas y respuestas con un diseño limpio y estructurado. Aunque parece sencillo, la clave está en los detalles: márgenes, paddings, alineaciones y colores bien ajustados.

**Detalles del proyecto**

<ins>Estructura y diseño</ins>

✔ Nav superior con los textos tal y como aparecen en el mockup.

✔ Tarjeta central con sombra, respetando la estructura del diseño.

✔ Botones funcionales con el estilo del diseño propuesto.

✔ Cambio de color en las opciones de la tarjeta al pasar el ratón.

✔ Márgenes, paddings y colores aproximados al diseño.

✔ Fuentes de Google Fonts para los textos.

✔ Iconos de Font Awesome para mejorar la apariencia.



<ins>Código y buenas prácticas</ins>

✔ Usar Flexbox o Grid para alinear los elementos correctamente.

✔ Crear solo 2 archivos: index.html y styles.css.

✔ El CSS debe contener al menos 10 reglas.

✔ Validar el HTML y CSS con el plugin "W3C Validation" en Visual Studio Code.
